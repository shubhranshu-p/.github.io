# Shubhranshu Sudeepta Panda — Portfolio

Personal portfolio for Shubhranshu Sudeepta Panda, a Java backend developer focused on Spring Boot, REST APIs, Spring Data JPA, and MySQL.

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production build

```bash
npm run build
```

The static website is generated in `out/`. Pushes to `main` are automatically published through GitHub Pages.
